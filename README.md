# CodeAlpha_age-calculator
A modern and interactive age calculator built with HTML, CSS, and JavaScript that provides accurate age insights through an elegant user interface. Beyond calculating age in years, the application offers detailed time statistics, birthday countdown tracking, birth day identification, and zodiac sign detection, all presented within a responsive and visually engaging design.

Features
Accurate age calculation in years
Total months, weeks, and days lived
Next birthday countdown
Birth day of the week detection
Zodiac sign identification
Responsive design for desktop and mobile devices
Animated starry background effects
Input validation to prevent future date selection
Clean and intuitive user experience
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Project Overview

 Age Calculator is a frontend web application designed to demonstrate practical JavaScript date manipulation, DOM interactions, responsive UI development, and modern web design principles. The project combines functionality with aesthetics to create an engaging user experience.
